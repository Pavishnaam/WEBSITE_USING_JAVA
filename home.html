Hospital Management System is an organized computerized
system designed and programmed to deal with day to day operations
and management of the hospital activities.
The project Hospital Management system includes registration of patients, storing their
details into the system, and also computerized billing in the pharmacy, and labs.
The interface is very user-friendly.
Hospital Management System is powerful, flexible, and easy to use and is designed and
developed to deliver real conceivable benefits to hospitals.
HMS provides the ability to manage all the paperwork in one place,
reducing the work of staff in arranging and analyzing the paperwork of the patients.

Elements used :
1.HTML ( for designing website )
2. CSS ( and style to the website )
3. JAVA ( for servlet programming )
4. MYSQL ( for accessing the data)
